# Dotfiles

### Create symbolic link
```
$ ln -s ~/.dotfiles/vim ~/.vim
$ ln -s ~/.dotfiles/vim/vimrc ~/.vimrc
$ ln -s ~/.dotfiles/tmux.conf ~/.tmux.conf
```
